Projects

Autonomous Car I built an autonomous car that can navigate a maze using computer vision and machine learning. The car is equipped with a camera, a Raspberry Pi, and a motor controller. The camera is used to capture images of the maze, and the Raspberry Pi is used to process the images and control the motor controller. The machine learning algorithm is used to identify the walls of the maze and the path to the exit. The car is able to navigate the maze autonomously and reach the exit without any human intervention. The application is designed to help users save money and get a better understanding of their financial situation. Mobile App I built a mobile app that allows users to learn about different programming languages. The app is built using Flutter and Dart. The app also provides users with practice exercises to help them learn the languages. The app is designed to help users learn about programming languages and improve their coding skills.
